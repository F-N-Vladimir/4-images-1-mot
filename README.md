# 4-images-1-mot

Jeu "4 Images 1 Mot" développé en Java avec NetBeans.

## Fonctionnalités
- Deviner le mot à partir de 4 images
- Système de niveaux
- ...

## Comment jouer
1. Ouvrir le projet dans NetBeans
2. Exécuter la classe principale `Main.java` (ou le nom de ta classe principale)

## Technologies
- Java
- NetBeans
- Swing / JavaFX (précise selon ce que tu as utilisé)
